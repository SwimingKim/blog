Set Homepage

이번에는 Home 페이지로 렌더링 할 노션 페이지를 설정하겠습니다.
 
먼저 홈 페이지로 사용할 노션 페이지의 ID를 찾아야 합니다. 복제한 Notbook 노션에서 Home 페이지로 사용할 페이지로 들어간 다음 주소에 표시된 페이지 아이디를 복사합니다.
 
복사한 페이지 아이디를 프로젝트 config/config.json의 homePageID 에 붙여넣어 줍니다.
 
홈 페이지 설정이 완료되었습니다.
